# Travelgram

Projeto desenvolvido durante os estudos de HTML5 e CSS3 com o objetivo de praticar a construção de layouts modernos, organização modular dos estilos e uso de Flexbox para criação de interfaces responsivas.

## Acesse o projeto
https://mariasmorais.github.io/Projeto-travelgram/

## Sobre o projeto

O projeto consiste em uma página de perfil de viagens inspirada em redes sociais, chamada Travelgram.

## A interface simula um perfil de usuário que compartilha fotos e experiências ao redor do mundo, contendo:

- Menu de navegação com ícones e links
- Cabeçalho com perfil do usuário e bio
- Informações como localização, número de países visitados e fotos
- Galeria de imagens organizada em grid
- Rodapé com informações institucionais

O desenvolvimento teve como foco a organização do código, HTML semântico, estilização moderna e boas práticas de estruturação de layouts com CSS.

## Tecnologias utilizadas:

- HTML5
- CSS3
- Flexbox
- CSS Grid (na organização da galeria)
- Variáveis CSS
- Organização modular dos arquivos CSS
- Boas práticas de estruturação de layout

## Funcionalidades: 

- Layout de perfil com foto e descrição
- Menu de navegação superior
- Galeria de fotos responsiva com flex-wrap
- Estilização de avatar com border-radius: 50%
- Organização modular dos estilos CSS
- Uso de variáveis CSS para cores e tipografia
- Estrutura semântica com header, nav, main e footer

## Como executar o projeto

Utilize o VS Code para executar o projeto utilizando a extensão Live Server.
